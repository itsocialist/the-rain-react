# The Rain Continues

> Browser-Native 3D | React Three Fiber | Neural Adaptive Systems

A browser-native 3D game built on **React Three Fiber** and **Three.js**, with neural network systems powered by **TensorFlow.js** and **Brain.js**. No downloads, no installs — just a URL.

## Quick Start

```bash
npm install
npm run dev
```

## Tech Stack

| Layer | Technology |
|---|---|
| **Rendering** | React Three Fiber + Three.js |
| **Physics** | Rapier (WASM) |
| **AI/ML** | TensorFlow.js + Brain.js |
| **State** | Zustand |
| **Audio** | Howler.js |
| **Build** | Vite |
